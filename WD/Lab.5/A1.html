Concept of WWW

  • It stands for World Wide Web.
  • It is basically a system of servers that support specially formatted documents.
  • It is a collection of text pages,videos,digital photographs and animations we can access over the internet.
  • User access the World Wide Web facilities via client called web browser.

HTTP Protocol

  1. HTTP stands for Hyper Text Transfer Protocol.
  2. HTTP is based on client-server architecture model and a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
